# Contributing to BotDev Community

Thank you for your interest in contributing! We welcome contributions from developers of all skill levels.

## How to Contribute

### Adding Resources to awesome-bot-resources

1. **Fork the repository**
2. **Add your resource** in the appropriate category
3. **Follow the format**: 
```markdown
   - [Resource Name](https://link-to-resource.com) - Brief description (1-2 sentences)
```
4. **Ensure quality**:
   - Resource must be actively maintained
   - Link must be working
   - Description must be clear and accurate
   - No duplicate entries
5. **Submit a pull request**

### Submitting a Starter Template

1. Create a repository with:
   - Clear README with setup instructions
   - Working example code
   - Dependencies list (requirements.txt, package.json, etc.)
   - LICENSE file (preferably MIT)
   - .gitignore
2. Test that it works from scratch
3. Submit an issue or PR linking to your template

### Reporting Issues

- Use GitHub Issues
- Provide clear description
- Include steps to reproduce (if applicable)
- Tag appropriately (bug, enhancement, question)

### Code of Conduct

- Be respectful and constructive
- Welcome newcomers
- Give credit where due
- Focus on collaboration

## Resource Quality Guidelines

Resources should be:
- **Active**: Updated within the last year
- **Documented**: Clear README and examples
- **Accessible**: Free or has free tier
- **Relevant**: Directly related to bot development

## Questions?

Open a [Discussion](https://github.com/orgs/botdev-community/discussions) or create an Issue.
